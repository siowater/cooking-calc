# ビルドと修正

TypeScriptおよびビルドエラーを段階的に修正します：

1. ビルドを実行する: npm run build または pnpm build
2. エラー出力を解析する:
* ファイルごとにグループ化する
* 重大度順に並べ替える


3. 各エラーについて:
* エラーのコンテキスト（前後5行）を表示する
* 問題を説明する
* 修正を提案する
* 修正を適用する
* ビルドを再実行する
* エラーが解決されたことを検証する


4. 以下の場合に停止する:
* 修正が新しいエラーを導入する場合
* 同じエラーが3回の試行後も持続する場合
* ユーザーが一時停止をリクエストする場合


5. サマリーを表示する:
* 修正されたエラー
* 残っているエラー
* 導入された新しいエラー



安全のために一度に一つのエラーを修正してください！
