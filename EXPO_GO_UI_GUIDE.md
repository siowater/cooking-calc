# Expo Goアプリの使い方（UIガイド）

## Expo Goアプリの画面構成

Expo GoアプリのUIはバージョンによって異なる場合があります。以下の方法を試してください。

## 方法1: ホーム画面からスキャン

### 最新バージョンのExpo Go（推奨）

1. **Expo Goアプリを起動**
2. **ホーム画面の上部**を確認
   - 「Scan QR code」ボタンがある場合：それをタップ
   - 「+」ボタンがある場合：それをタップ
   - 「Connect to development server」がある場合：それをタップ

### 画面の見つけ方

ホーム画面で以下のいずれかを探してください：
- 「Scan QR code」
- 「QRコードをスキャン」
- 「+」（プラスボタン）
- 「Connect」
- 「Connect to development server」

## 方法2: URLを手動入力（最も確実）

「Scan QR code」ボタンが見つからない場合、URLを手動で入力する方法が最も確実です。

### ステップ1: ターミナルでURLを確認

`npm start`を実行しているターミナルで、以下のような行を探してください：

```
› Metro waiting on exp://192.168.x.x:8083
```

または

```
exp://192.168.1.100:8083
```

この`exp://`で始まるURLをコピーします。

### ステップ2: Expo GoアプリでURLを入力

1. **Expo Goアプリを開く**
2. **ホーム画面で「Enter URL manually」または「URLを手動入力」を探す**
   - 見つからない場合、ホーム画面の下部や設定画面を確認
3. **URLを入力**
   - ターミナルからコピーした`exp://`で始まるURLを貼り付け
   - 例: `exp://192.168.1.100:8083`

## 方法3: 最近のプロジェクトから選択

Expo Goアプリのホーム画面に「Recent projects」や「最近のプロジェクト」セクションがある場合：

1. そのセクションを確認
2. 以前に接続したプロジェクトがあれば、それをタップして再接続

## 方法4: 検索バーを使用

Expo Goアプリに検索バーがある場合：

1. 検索バーをタップ
2. ターミナルのURLを入力
3. 接続をタップ

## 画面が見つからない場合の確認事項

### 1. Expo Goアプリのバージョンを確認

- App Store/Play Storeで最新バージョンに更新されているか確認
- 古いバージョンを使用している場合、更新してください

### 2. アプリの権限を確認

- カメラの権限が許可されているか確認
- 設定アプリからExpo Goの権限を確認

### 3. 別の方法を試す

URLを手動入力する方法が最も確実です。

## 手動入力の詳細手順

### ステップ1: ターミナルでURLを確認

`npm start`を実行しているターミナルで、以下のような行を探してください：

```
› Metro waiting on exp://192.168.x.x:8083
```

**このURLをコピーします。**

### ステップ2: Expo GoアプリでURLを入力

1. **Expo Goアプリを開く**
2. **ホーム画面のどこかに「Enter URL manually」や「URLを手動入力」があるはずです**
   - 見つからない場合、画面をスクロールして確認
   - または、ホーム画面の上部の検索バーをタップ
3. **URLを貼り付け**
   - ターミナルからコピーしたURLを貼り付け
   - 例: `exp://192.168.1.100:8083`
4. **「Connect」または「接続」をタップ**

## トラブルシューティング

### URLが見つからない場合

ターミナルにURLが表示されていない場合：

1. `npm start`を実行しているターミナルを確認
2. QRコードの下にURLが表示されているはずです
3. 表示されていない場合、`npx expo start --clear`で再起動

### 接続できない場合

1. **PCとスマートフォンが同じWi-Fiネットワークに接続されているか確認**
   - これが最も重要なポイントです
2. **PCのIPアドレスを確認**
   - Windowsの場合: `ipconfig`コマンドで確認
   - 表示されたIPアドレスがURLのIPアドレスと一致しているか確認
3. **ファイアウォールの設定を確認**
   - Windowsファイアウォールがポートをブロックしていないか確認

## まとめ

- 🔍 **「Scan QR code」ボタンが見つからない場合、URLを手動入力する方法が最も確実です**
- 📋 **ターミナルに表示されている`exp://`で始まるURLをコピー**
- ✅ **Expo Goアプリで「Enter URL manually」を探してURLを入力**
- 🌐 **PCとスマートフォンが同じWi-Fiネットワークに接続されていることを確認**
