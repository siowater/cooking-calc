# Expoアプリの起動方法

## 起動コマンド

Cursorのターミナルで以下のコマンドを実行してください：

```bash
npm start
```

または、ポート8083を明示的に指定する場合：

```bash
npx expo start --clear --port 8083
```

## QRコードの確認

1. コマンド実行後、数秒待つとQRコードが表示されます
2. QRコードはターミナルウィンドウに直接表示されます（ログファイルには含まれません）
3. QRコードの下に以下のような情報が表示されます：
   ```
   › Metro waiting on exp://192.168.x.x:8083
   › Scan the QR code above with Expo Go (Android) or the Camera app (iOS)
   ```

## ポート番号の確認

QRコードのURLに含まれるポート番号を確認してください：
- `exp://192.168.x.x:8083` のように表示されます
- ポート8083で起動している場合、QRコードも8083用になります

## トラブルシューティング

### QRコードが表示されない場合

1. ターミナルウィンドウをスクロールして確認
2. 数秒待ってから再度確認
3. `Ctrl+C`で停止してから、再度`npm start`を実行

### ポートが異なる場合

- ターミナルに表示されるURLを確認
- `exp://192.168.x.x:XXXX` のXXXX部分がポート番号です
