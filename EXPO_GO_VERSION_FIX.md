# Expo Goバージョン互換性エラーの解決方法

## エラーメッセージ

"project is incompatible with this version of expo go"

このエラーは、**Expo Goアプリのバージョンが古い**ことを示しています。

## 現在のプロジェクトの設定

このプロジェクトは **Expo SDK 51** を使用しています。

Expo SDK 51を使用するには、**最新バージョンのExpo Goアプリ**が必要です。

## 解決方法

### 方法1: Expo Goアプリを更新（推奨）

#### iOSの場合

1. **App Storeを開く**
2. **「更新」タブを開く**
3. **「Expo Go」を探す**
4. **「更新」ボタンをタップ**

または、App Storeで「Expo Go」を検索して、更新ボタンがあるか確認してください。

#### Androidの場合

1. **Google Play Storeを開く**
2. **「マイアプリとゲーム」を開く**
3. **「Expo Go」を探す**
4. **「更新」ボタンをタップ**

または、Google Play Storeで「Expo Go」を検索して、更新ボタンがあるか確認してください。

### 方法2: Expo Goアプリを再インストール

更新ボタンが表示されない場合、アプリを再インストールしてください：

1. **Expo Goアプリを削除**
2. **App Store/Play Storeから最新版を再インストール**

### 方法3: Expo SDKバージョンを確認

プロジェクトのExpo SDKバージョンを確認：

```bash
npx expo --version
```

または、`package.json`を確認：

```json
"expo": "~51.0.0"
```

## Expo SDKとExpo Goの互換性

| Expo SDK | Expo Goアプリの最小バージョン |
|----------|---------------------------|
| SDK 51   | 最新版（2024年後半以降）    |
| SDK 50   | 最新版（2024年後半以降）    |
| SDK 49   | 2024年後半以降             |

## 確認手順

### ステップ1: Expo Goアプリのバージョンを確認

1. **Expo Goアプリを開く**
2. **設定画面を開く**（通常はアプリの下部にあります）
3. **バージョン情報を確認**

### ステップ2: 最新バージョンか確認

- **iOS**: App Storeで「Expo Go」を検索して、最新バージョンか確認
- **Android**: Google Play Storeで「Expo Go」を検索して、最新バージョンか確認

### ステップ3: 更新後、再度接続

1. **Expo Goアプリを更新**
2. **アプリを再起動**
3. **再度QRコードをスキャン、またはURLを入力**

## トラブルシューティング

### 更新ボタンが表示されない場合

1. **App Store/Play Storeを再起動**
2. **アプリを削除して再インストール**
3. **デバイスを再起動**

### 更新後もエラーが表示される場合

1. **Expo Goアプリを完全に再起動**（アプリを閉じて再度開く）
2. **ターミナルで`npm start`を再起動**
3. **キャッシュをクリア**:
   ```bash
   npx expo start --clear
   ```

### それでも解決しない場合

プロジェクトのExpo SDKバージョンを確認：

```bash
npx expo --version
```

もしExpo SDK 51が最新でない場合、Expo Goアプリが対応していない可能性があります。

## まとめ

- 🔄 **Expo Goアプリを最新バージョンに更新してください**
- ✅ **更新後、アプリを再起動して再度接続してください**
- 📱 **App Store/Play Storeから更新できます**

## 次のステップ

1. **Expo Goアプリを更新**
2. **アプリを再起動**
3. **再度QRコードをスキャン、またはURLを入力**

更新後もエラーが表示される場合は、ターミナルのエラーメッセージを共有してください。
