# エラーログの確認方法

## ⚠️ 重要な注意点

**カメラアプリからQRコードをスキャンしても動作しません！**

Expo Goアプリ**内**の「Scan QR code」ボタンからスキャンする必要があります。

## エラーログの確認場所

### 1. ターミナル（Metro Bundlerのログ）⭐最重要

**エラーログは主にターミナルに表示されます。**

`npm start`を実行しているターミナルウィンドウを確認してください：

```
ERROR  Error: ...
ERROR  Unable to resolve module ...
ERROR  Cannot find module ...
```

**確認方法**:
1. `npm start`を実行しているターミナルウィンドウを開く
2. エラーメッセージが赤い文字で表示されているか確認
3. `ERROR`で始まる行を探す
4. エラーメッセージ全体をコピーして共有してください

### 2. Expo Goアプリ内のエラーメッセージ

Expo Goアプリでエラーが発生した場合、アプリ内にエラーメッセージが表示されます：

- 「Unable to connect」
- 「there was a problem running the requested app」
- 「Unknown error」

**確認方法**:
1. Expo Goアプリを開く
2. エラーメッセージが表示されているか確認
3. エラーメッセージの全文をコピー

### 3. デバイスのログ（Android）

Androidデバイスの場合、`adb logcat`でログを確認できます：

```bash
adb logcat | grep -i expo
```

### 4. デバイスのログ（iOS）

iOSデバイスの場合、XcodeのConsoleでログを確認できます。

## 正しいQRコードスキャン方法

### ❌ 間違った方法

- **カメラアプリからスキャン**: これは動作しません
- **他のQRコードスキャンアプリからスキャン**: これも動作しません

### ✅ 正しい方法

1. **Expo Goアプリを起動**
   - App Store/Play Storeから「Expo Go」をインストール
   - アプリを起動

2. **アプリ内の「Scan QR code」ボタンをタップ**
   - Expo Goアプリのホーム画面の上部に「Scan QR code」ボタンがあります
   - このボタンをタップすると、カメラが起動します

3. **ターミナルのQRコードをスキャン**
   - PCのターミナルに表示されているQRコードをスキャン
   - または、URLを手動入力することも可能

## エラーログの取得手順

### ステップ1: ターミナルのログを確認

1. `npm start`を実行しているターミナルウィンドウを開く
2. エラーメッセージを探す（赤い文字で表示）
3. エラーメッセージ全体をコピー

### ステップ2: Expo Goアプリのエラーメッセージを確認

1. Expo Goアプリを開く
2. エラーメッセージが表示されているか確認
3. エラーメッセージの全文をコピー

### ステップ3: デバッグモードを有効化（オプション）

Expo Goアプリで：
1. デバイスをシェイク（振る）
2. 「Debug Remote JS」を選択
3. Chrome DevToolsが開き、より詳細なエラー情報を確認できます

## よくあるエラーと対処法

### エラー1: "Unable to connect"

**原因**: PCとスマートフォンが同じWi-Fiネットワークに接続されていない

**対処法**:
1. PCとスマートフォンが同じWi-Fiに接続されているか確認
2. ファイアウォールの設定を確認
3. URLを手動入力してみる

### エラー2: "there was a problem running the requested app"

**原因**: アプリのコードにエラーがある

**対処法**:
1. ターミナルのエラーログを確認
2. `npx expo start --clear`でキャッシュをクリア
3. `npm install --legacy-peer-deps`で依存関係を再インストール

### エラー3: "Unknown error"

**原因**: 詳細なエラー情報がターミナルに表示されている可能性が高い

**対処法**:
1. ターミナルのログを確認
2. エラーメッセージ全体をコピーして共有

## デバッグ用コマンド

### キャッシュをクリアして再起動

```bash
npx expo start --clear
```

### 依存関係を再インストール

```bash
npm install --legacy-peer-deps
```

### ポートを指定して起動

```bash
npx expo start --clear --port 8083
```

## エラーログを共有する際の情報

以下の情報を含めてください：

1. **ターミナルのエラーメッセージ**（全文）
2. **Expo Goアプリのエラーメッセージ**（全文）
3. **実行したコマンド**（`npm start`など）
4. **デバイスの種類**（iOS/Android）
5. **ネットワーク環境**（同じWi-Fiに接続されているか）

## まとめ

- ⚠️ **カメラアプリからスキャンしても動作しません**
- ✅ **Expo Goアプリ内の「Scan QR code」ボタンからスキャンしてください**
- 📋 **エラーログは主にターミナルに表示されます**
- 🔍 **エラーメッセージ全体をコピーして共有してください**
